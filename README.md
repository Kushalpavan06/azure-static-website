# Azure Static Website Hosting

## 🚀 Project Overview
Deployed a static portfolio website using Azure Storage Static Website feature.

## 🧰 Technologies Used
- HTML, CSS
- Azure Storage Account

## ⚙️ Steps
1. Created storage account
2. Enabled static website
3. Uploaded files to $web container
4. Accessed via public endpoint
